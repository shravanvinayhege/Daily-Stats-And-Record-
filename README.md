# Daily-Stats-And-Record-
A Python-based sales management app with GUI and SQLite backend. Calculates, stores, and analyzes company data locally with a clean interface and efficient backend logic
#Requirements
To run this project, make sure you have:
• 	Python 3.8+ installed
• 	SQLite3 (comes bundled with Python, but ensure it’s available)
• 	Tkinter (for GUI; usually included with Python, but verify installation)
